# 代码性能和安全检查器

## 功能

- **性能分析**：识别代码中的性能瓶颈，包括：
  - 循环优化机会
  - 内存使用问题
  - 算法复杂度分析
  - 资源泄露风险

- **安全检查**：检测常见的安全漏洞，包括：
  - SQL 注入风险
  - XSS 攻击风险
  - 敏感信息泄露
  - 权限控制问题
  - 加密实现缺陷

- **代码质量评估**：
  - 代码复杂度分析
  - 代码风格检查
  - 潜在的逻辑错误
  - 最佳实践建议

## 使用方法

1. **分析整个项目**：当用户要求对整个项目进行性能和安全检查时
2. **分析特定文件**：当用户指定特定文件进行检查时
3. **分析特定功能**：当用户要求分析特定功能的性能或安全问题时
4. **预处理检查**：在代码提交前进行检查，确保代码质量

## 输出内容

- 性能问题列表及优化建议
- 安全漏洞列表及修复方案
- 代码质量评分及改进建议
- 具体问题的代码位置和详细说明

## 示例使用场景

- "请检查这个项目的性能和安全问题"
- "分析这个文件中的性能瓶颈"
- "检测代码中的安全漏洞"
- "评估这段代码的质量"
- "在提交前检查代码质量"
