# GitFame

## 欢迎使用 GitFame

GitFame 是一个功能强大的 Git 仓库分析工具，帮助你了解代码贡献者的贡献情况。

### 项目分类

- **一级分类**：开发利器 (DevTools)
- **二级分类**：终端增强 (CLI)
- **收录侧重**：提升效率百分比，强调对工作流的改造

### 主要特性

- 📊 **贡献者分析**：详细的贡献者统计信息
- 🔍 **代码热度**：分析代码修改频率
- 📈 **趋势图表**：直观展示项目发展趋势
- 🎯 **精确统计**：准确计算代码贡献量
- ⚡ **效率提升**：相比手动分析，提升效率达 95% 以上

### 快速开始

1. **安装 GitFame**
2. **运行分析**
3. **查看报告**

## 如何使用

```bash
# 安装
npm install -g gitfame

# 分析仓库
gitfame /path/to/repository

# 查看报告
gitfame --report /path/to/repository
```
